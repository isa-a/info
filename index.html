MSc project: Stochastic modelling of bacterial interactions in the gut microbiota

Repository: https://github.com/isa-a/mscproject

See the full paper .

Aim of study: Use stochastic differential equations[1] and Approximate Bayesian Computation[2] to create a program that can estimate the values of parameters representing competition between bacterial species in the human gut

Example: Two species of bacteria in the gut compete with each other, species A competes on species B with an effect of 0.3, and species B competes on species A with an effect of 0.6 (αAB=0.3 and αBA=0.6) The program then estimates these values to be:

Sample means: alphaAB: 0.302, alphaBA: 0.607 

The Engine for Likelihood-Free Inference (ELFI)[3] allows us to plot the discrepancies for each estimation:

 

 and plot the estimations as a distribution:

 

 

Potential future work:

  • Extend ABC methods to fit more parameters so an n number of species can be passed through at one time
  • Fit more experimental data to make inferences
  • Apply GUI to program to make it interactive and easier to use
  • Use in conjunction with new lab experiment data
  • Application to the epidemiology of Clostridioides difficile e.g. investigate species of bacteria which can outcompete C. difficile
